# OpenEVmap

**OpenEVmap** is an open-source tool for mapping and maintaining EV charging stations in OpenStreetMap.

It helps contributors, EV operators, and developers add and edit EV charging infrastructure data correctly, consistently, and with respect to the OpenStreetMap ecosystem.

All edits are made directly to OpenStreetMap.

## ğŸ§  Relationship to commercial products

OpenEVmap is an **open, community-oriented project**.

It is developed alongside a commercial EV product, but:
- OpenEVmap itself is not a closed or paid service
- it does not restrict access to data
- it improves shared infrastructure for everyone

Commercial products may build **on top of** improved OSM data, but OpenEVmap always pushes data back to OSM.
